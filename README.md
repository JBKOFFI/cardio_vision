## Projet
Application interactive développée par :

Lien vers l'application : [CardioVision](https://jblalegende17.shinyapps.io/projet2025_JB_la_legende/)

## Contexte
Les maladies cardiovasculaires représentent près de 18 millions de décès par an dans le monde. CardioVision vise à explorer les facteurs de risque et à prédire le potentiel risque d’attaque cardiaque à partir de données médicales.

## Données
- **Cliniques** : 303 patients avec facteurs mesurables (âge, cholestérol, pression, etc.) et non mesurables (sexe, type de douleur, ECG, etc.).  
- **Géographiques** : Données OWID sur les maladies cardiovasculaires (2000-2021).

## Fonctionnalités
1. **Cartographie** : visualisation interactive mondiale des décès cardiovasculaires.  
2. **Exploration** : exploration dynamique des données et graphiques sur les facteurs de risque.  
3. **Comprendre** : analyse et comparaison de modèles prédictifs (logistique, Random Forest) avec critères de performance (AUC, sensibilité, spécificité, odds ratio).  
4. **Prévenir** : formulaire interactif pour estimer le risque cardiaque potentiel et fournir des recommandations personnalisées.

## Modèles prédictifs
- Modèle complet et modèle réduit (sélection de variables significatives)  
- Random Forest et régression logistique utilisées pour prédictions et analyses

## Futur
- Intégration de modèles supplémentaires (ADL, forêts d’arbres)  
- Version mobile  
- Ajout de données médicales actualisées  
- Collaboration avec des professionnels de santé

## Objectif
CardioVision combine science, visualisation et accessibilité pour prévenir et comprendre les risques cardiovasculaires à des fins pédagogiques et exploratoires.
